Self-identification 
7.1 
The maritime identity as defined in Recommendation ITU-R M.585, coded as indicated 
in § 5.2 and its Note 1, is used for self-identification. 
8 
Messages 
The messages that are included in a call sequence contain the following message elements, which are 
listed in the order in which they would appear in each message. All message formats are explicitly 
defined in Tables A1-4.1 through A1-4.11. The procedure to compose digital selective calling 
message should comply to § 3.2.3.4 of Annex 4. 
8.1 
For a “distress” alert (see Table A1-4.1) the distress information is contained in four 
messages in the following order:
